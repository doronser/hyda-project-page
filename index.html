HyDA: Hypernetworks for Unsupervised Domain Adaptation in Medical Imaging

Doron Serebro, Tammy Riklin-Raviv,
Ben-Gurion University of the Negev
MICCAI 2025
Paper (Coming Soon) Code (Coming Soon) arXiv

We present HyDA, a novel hypernetwork-based framework for unsupervised domain adaptation in medical imaging. HyDA learns implicit domain representations which are passed to a hypernetwork that generates domain-specific parameters for the primary network.

Abstract

Domain shift is a significant challenge in adapting medical imaging analysis models for practical use. Medical imaging datasets often exhibit variations due to differences in acquisition protocols, patient demographics, and imaging devices. Current approaches often struggle with generalization across diverse datasets, leading to decreased performance when models trained on data from specific sources are applied to others. These limitations hinder the widespread deployment of models in clinical settings, where data variability is inevitable. In this work, we introduce HyDA, a novel hypernetwork-based framework that leverages domain information instead of ignoring it, enabling dynamic adaptation at inference time. Unlike prior UDA methods that only utilize domain priors during training, HyDA learns implicit domain representations and uses them to adjust model parameters on-the-fly, making it the first approach to provide real-time adaptation to unseen medical domains. We validate HyDA on two clinically relevant tasks — MRI brain age prediction and chest X-ray pathology classification — demonstrating its ability to generalize across tasks and modalities.

Main Concept

Most unsupervised domain adaptation (UDA) methods use domain prior as a negative signal in an attempt to align the source and target distributions. This results in a domain-agnostic feature space, which essentially erases any domain information. In this work, we propose an alternative approach, where instead of aiming to learn a single global domain-agnostic feature space, we rely on domain specific features to dynamically adapt the model for each sample on-the-fly. This is done using a separate domain branch and a hypernetwork that generates domain-specific parameters for the primary network.

Brain MRI Age Prediction

For the brain age prediction task, we used a custom 3D CNN architecture. The domain branch is similar to the primary network, but with significantly less parameters. The hypernework is a 2 layer MLP that projects domain embedding to an intermediate space, and an output layer that generates domain-specific parameters for the age regressor in the primary network.
